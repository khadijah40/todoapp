##  How It Works

```
User types a task and submits
           ↓
app.js creates a new list item in the DOM
           ↓
Task is saved to localStorage
           ↓
User clicks a task → toggles complete state (checked/unchecked icon)
           ↓
User clicks delete → removes task from DOM and localStorage
           ↓
On page reload → tasks are loaded back from localStorage
```

## Getting Started

No installs, no setup, no terminal commands needed.

### 1. Clone the repository

```bash
git clone https://github.com/khadijah40/todoapp.git
```

### 2. Open in your browser

```bash
cd todoapp
open index.html
```

Or simply double-click `index.html` — it opens directly in any browser and is ready to use.
